# Phase 2 Evidence Packet Templates

These templates support Phase 2 evidence preparation only.

They do not approve any human decision, do not freeze any pack, and do not define real workload membership.

Use them to prepare reviewer-robust evidence packets for:

- `HD-03` anchor scale-factor approval
- `HD-04` TPC-DS slice approval
- `HD-05` JOB slice approval
- `HD-06` public-realistic shortlist approval
- `HD-07` semantic-risk final-set approval

Each packet should be backed by:

- a human-readable decision memo from this directory
- one or more machine-readable manifests under `results/templates/`
- later, actual run outputs under `results/`
- later, summary tables under `analysis/`

Nothing in this directory is itself an approval artifact.
